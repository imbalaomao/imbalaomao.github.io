【科学上网】个人推荐的软件

· 2026-01-10 · # 推荐

本推荐从 海豚机场测速 结合本人使用情况修改而来，不求多但求精，如需要多软件对比，请查看原文章。

🪟 Windows

FlClash

  • 优点：
    • ✅Flutter 编写
    • ✅UI 简单易上手
    • ✅运行轻量稳定
    • ✅完整覆盖基础功能和基本覆写（仅支持 javascript 语言）
    • ✅自带配置编辑器、DNS 覆写、规则覆写（图形化添加规则）
    • ✅支持 WebDAV 配置多端同步功能
  • 缺点：
    • ❌UI风格偏手机化，部分操作反桌面端直觉（如策略组过多时，需拖动切换策略组，或点击最右侧小箭头才可快速切换）
    • ❌无法替换内核（采用 flclash 自编译内核）
    • ❌不支持 Smart 规则组（由于原版 Mihomo 内核不支持）
    • ❌不支持 YAML 覆写（但可以让 AI 将 YAML 覆写修改为 JS 格式）
    • ❌不支持便携化（默认配置存放在 %appdata%\com.follow\clash ）

下载地址：https://github.com/chen08209/FlClash/releases/latest

Clash Mi

  • 优点：
    • ✅小白友好软件（无覆写功能，主打开箱导入即用）
    • ✅支持便携模式（配置文件存放在程序目录中，可以打包带走）
  • 缺点：
    • ❌无法直观看到各策略组对应使用节点情况
    • ❌不支持覆写功能，想自定义规则只能采用节点转换工具
    • ❌无法替换内核（内核与主程序写死在一起）

下载地址：https://github.com/KaringX/clashmi/releases/latest

🤖 Android 安卓

FlClash

  • 优点：
    • ✅UI简单易上手
    • ✅运行轻量稳定
    • ✅支持 Javascript 语言覆写功能（相对于 CMFA 等老牌软件）
    • ✅自带配置编辑器、DNS覆写、规则覆写（图形化添加规则）
    • ✅支持 WebDAV 配置多端同步功能，可实现电脑修改配置、手机同步应用
  • 缺点：
    • ❌不支持 YAML 覆写（但可以让 AI 将 YAML 覆写修改为 JS 格式）

下载地址：https://github.com/chen08209/FlClash/releases/latest

🍎 iOS

Loon

  • 优点：
    • ✅生态较为良好，去广告插件齐全（但主要为可莉大佬编写）
    • ✅拥有“逻辑分流规则（AND/OR）”，对于复杂规则兼容性强
    • ✅UI较为直观，许多配置可通过图形化界面直接配置
  • 缺点：
    • ❌多数插件较为封闭，无法用于其他软件中（lpx后缀）
    • ❌相对QX、Shadowrocket等软件较容易出现断流问题（尤其是开启IPV6后），目前无解
    • ❌对于新协议（AnyTLS、XHTTP等）跟进速度慢，适合机场党
    • ❌价格相对较高（$7.99），对比Shadowrocket无明显

购买地址：https://apps.apple.com/app/loon/id1373567447

Shadowrocket

  • 优点：
    • ✅新手最友好的软件，上手非常简单
    • ✅最便宜的 iOS 代理工具（$2.99），且支持多人共享
    • ✅代理协议追新速度最快，已支持 XHTTP、AnyTLS 等新协议
    • ✅兼容大部分 Surge 模块
    • ✅可通过导入配置等方式修改软件皮肤
  • 缺点：
    • ❌模块规则设置逻辑略反直觉（尤其是配置中包含增强脚本的情况下）
    • ❌多配置 MITM 无法共用证书，需重新配置
    • ❌设置较为繁琐，每一步做完都需要转一会圈，因此建议自行改写好配置后直接导入，不然调整设置很累
    • ❌模块和插件并不算丰富，且一些 Surge 插件无法完美兼容

购买地址：https://apps.apple.com/app/shadowrocket/id932747118
