【技巧】Essential-PH1刷机方法

· 2023-03-26 · # 技巧

准备

  • 官方安卓P的090版本底包
  • adb工具
  • Essential官方驱动
  • 解锁OEM并打开ADB调试功能

步骤

  • 安装驱动
  • 下载工具并解压后，在控制台（推荐cmd，powershell也可以）中输入adb devices，查看手机是否能够被识别
  • 刷入安卓P 090底包（解压后双击flashall.bat即可）
  • 刷入twrp（或不刷入也可以）
  • 进入twrp，刷入系统和magisk（可以用adb sideload）

注意

根据这篇文章中的描述，由于安全固件的版本号对不上，在Opposite分区的Rec中屏幕会断触，官方也提到了这个问题，采用adb sideload方法即可：
先执行这行：adb shell twrp sideload
再执行这行：adb sideload 你准备刷入的zip文件名.zip

备选解决方案

刷入twrp img之后，使用adb sideload安装twrp installer，安装完成后进入bl将分区改为a区即可解决断触问题

# 查看你手机当前使用的分区
fastboot getvar current-slot
# 将分区转为B或A
fastboot --set-active=b（如果是a分区，将b改为a即可）

建议参考文章: 知乎

补充说明：

  • 解决开机红色感叹号的方法：在能使用adb的时候输入adb reboot "dm-verity enforcing"即可
  • 推荐刷入系统后紧接着刷入twrp-installer，重启后再刷入magisk，这样比较保险
  • 有关于刷入Magisk修改的BOOT之后卡在fastboot的拯救方法
