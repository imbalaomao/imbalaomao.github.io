前言

自2012年接触翻墙以来，我大约用过以下各类软件：

  • VPN系列：PPTP、L2TP类软件（类似GreenVPN等）
  • 法轮功成员开发软件：Lantern（蓝灯）、自由门、无界……（可能还有，但我就用过这几种）
  • 初代代理软件：Psiphon（赛风）、XX-NET（基于GAEAgent且自带一定混淆的软件，直至今天仍在更新）
  • 传奇代理软件：Shadowsocks(R)（可以说没有这款软件就没有今天的代理软件百家齐放的局面。R.I.P. Clowwindy&Breakwa11）
  • 现代代理软件：v2ray、xray、Clash、QuantumultX、Shadowrocket、Surge…
  • 小众代理软件：Tuic、Hysteria、Naiveproxy、mieru…

在这分享（记录）一下，方便后续查阅。

Windows GUI

Clash系

  • Clash for Windows（已删库，此链接为最终汉化备份版）
  • Clash for Windows汉化版
  • Clash Verge（需要EdgeRuntime2，轻量好用）
  • ClashWarden（使用C++编写的小巧客户端，许久不更新了）
  • ClashN（V2rayN作者做的同界面客户端，许久不更新了）
  • Fclash（使用Flutter编写的跨平台GUI）
  • clash-for-flutter（使用谷歌Flutter框架编写）
  • Clash.Mini（使用Go语言编写的简洁版UI，集成了Meta内核，许久不更新了）
  • ClashR Auto（基于CFW魔改）
  • Clashy（桌面全平台）

V2ray系

  • Nekoray（魔改Qv2Ray，增加hysteria等新协议支持）
  • V2rayN（老牌*ray系软件，近期已用.NET 6.0重写）
  • Qv2ray（原维护者弃坑，新仓库为RELEASE备份）

Sing-Box

  • Sing-box 托盘GUI

iOS/MacOS GUI

  • Shadowrocket(仅限iOS系统，不支持intel Mac，一般定价：2.99USD)
  • QuantumultX(iOS/MacOS授权通用，支持Intel-Mac/M1-Mac，一般定价：4.99USD)
  • Surge(iOS/MacOS授权分离，定价超高，土豪专用)
  • Loon(iOS/MacOS授权通用)
  • Stash(iOS可用，与Clash premium核心共享配置文件，2.99USD)
  • ClashX（仅限Mac使用）
  • ClashX Meta（仅限Mac使用）
  • V2rayX（仅限Mac使用）
  • Sing-box 托盘GUI（仅限Mac使用）

Android GUI

V2ray系

  • SagerNet（兼容SSR、v2ray、clash配置，带很多插件，已停更）
  • Matsuri（魔改版Sagernet，与原版差距不大，兼容原版插件）
  • V2rayNG（Xray社区钦定）

Clash系

  • Clash for Android
  • Clash Meta for Android
  • Clash for Magisk（需ROOT）

Surfboard（兼容Surge4配置文件）

路由器

  • fancyss(梅林/华硕固件经典插件)
  • MerlinClash(使用ClashPremium/ClashMeta内核)
  • OpenClash(OpenWRT经典插件)
  • ShellClash(只要能SSH就能Clash！)

搭建脚本归类

所有脚本我本人都使用过，无好坏之分，只有适合你自己的才是最好的

v2ray/xray脚本

  • 233boy的v2ray一键脚本
  • 233boy的xray一键脚本
  • mack-a 编写的八合一脚本，简单易用
  • 1024.day站长自编简易wss+ss安装脚本，脚本足够安全

sing-box脚本

  • fscarmen的一键脚本
  • FranzKafkaYu的一键脚本
  • yuehen7的一键脚本

Trojan脚本

请注意：Trojan原版核心已两年多未更新，Trojan-go核心已一年多未更新，因此存在被探测可能性，保稳定性推荐使用v2ray/xray脚本或使用下方的小众协议

  • 世界上最简单的Trojan一键部署脚本，无需域名申请（目前本人正在使用）
  • BBR + Trojan-go + 其他脚本合集
  • Overplus，IP裸连（基于trojan，域名sniffer为bing搜索，但客户端跳过证书验证，目前稳定性尚佳）
  • Jrohy的Trojan一键脚本，但卸载部分无法完全删除安装时产生的文件，不是很推荐

小众协议脚本/教程

本Part中很少会出现一键脚本，大多为分步骤教学，需要对Linux有一定基础知识。同时大多数小众协议并无图形化UI，因此请使用v2rayN或Nekoray添加自定义核心后使用，具体使用方法请参考此处

  • Naiveproxy安装脚本 by MisakaBlog
  • Hi-Hysteria一键安装脚本（未适配Hysteria 2.0）
  • Hysteria 2.0安装脚本 by MisakaBlog
  • Hysteria 2.0安装脚本 by Qin2DIM(本脚本依赖python 3.8+)
  • Tuic安装脚本 by MisakaBlog
  • mieru官方教程（分步骤且为英文，非脚本）
  • socks5代理搭建

小众协议补充：

  1. 目前对于如Racknerd、virmach这类无药可救的机器，可以使用Hysteria协议，如有CN2GIA、CU4837、CU9929等优秀线路的机器推荐优先考虑Tuic、Naiveproxy等有流控的协议（Hysteria无法搭建在Racknerd机器上）。
  2. Clash.meta目前已对Tuic、Hysteria做了较为完善的支持，不想添加自定义配置和核心的可以使用此软件（关于替换核心可以看本人教程）

机场

对于小白且对于翻墙IP要求低的用户来说，机场是最好的选择。但由于最近的 v2board群体脱库事件 导致很多人不敢买机场，以下推荐的机场都是没有使用v2board的，因此可以放心购买：

  • YToo（歪兔：极稳定、贵、速度很快）
  • Saiki（新开机场，低价大流量速度快）
  • X-air（稳定、快速、便宜）
