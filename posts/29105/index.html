赠送版 L1

配置

  • vCPU: 1 x AMD EPYC 7402P
  • 内存: DDR4, 3200Mhz => 1GB
  • 存储: NVMe, SSD, PCIe 4 => 20GB
  • 网络连接性:
    • 最高优先级，限制流量 (按预售购买机型计算*) => 1Gbps
    • 低优先级，与特定数量用户共享，不限制流量 => 100Mbps
  • 续费月费: $1 / ¥7

注:

  • 客户* : 指订单与机器在同一个帐户上，没有任何 Push 记录的客户为满足赠送条件
  • 按预售购买机型计算*: A 系列提供 750GB 高优先级月流量, P 系列提供 500GB 高优先级月流量

测评

正式版 L1

配置

  • vCPU 0.75x (EYPC 7xx2 - 均衡分配)
  • 内存 512M (可选配 $1 / GB 起, 最多 4 GB)
  • 系统盘 20G | NVMe SSD
  • 存储盘 无
  • 月流量 无限制*
  • 公网网络速率 1Gbps
  • IPv4 地址 0 x (可选购)
  • IPv6 地址 1 x /64
  • DDoS防御 无 (被攻击黑洞2小时)
  • 1Gbps @ 限定 300 GB 流量
  • 超出流量后降级 共享 100Mbps @ 无限流量
