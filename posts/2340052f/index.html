【技巧】利用Sublime Text破解Sublime Text并配置Hexo写作环境

  • 作者 OldcatY
  • 724 字

本文最后编辑于 前，其中的内容可能需要更新。

    1. 1.1. 下载Sublime Text
    2. 1.2. 破解

破解步骤

下载Sublime Text

官网： https://www.sublimetext.com

下载地址：https://download.sublimetext.com/sublime_text_build_4152_x64_setup.exe

破解

  1. 安装之后，使用sublime text 打开安装目录下的sublime_text.exe文件
  2. 搜到到 80 7805 000f 94c1
  3. 更改为 c6 4005 0148 85c9
  4. 然后另存到其他路径，然后关闭sublime text，将原sublime_text.exe进行替换即可

安装汉化

  1. 点击菜单栏Tools—>Install Package Control 跳出弹框点击 确定按钮
  2. 点击菜单栏 Preferences – Package Control
  3. 弹框中输入 install package，选中 Package Control: Install Package 然后等待弹窗(可能需要耐心等待一段时间)
  4. 等出现弹窗后, 在弹窗的搜索框中输入 ChineseLocalizations , 点击出现的选项
  5. 等待一段时间，如果出现了如下画面则汉化步骤全部完成
  6. 后续若需要更改，可点击菜单栏 帮助-–>Language 选择进行更改。

配置Hexo写作环境

安装Terminal插件

  1. 弹框中输入 install package，选中 Package Control: Install Package 然后等待弹窗(可能需要耐心等待一段时间)
  2. 等出现弹窗后，在弹窗的搜索框中输入 Terminal，点击出现的选项
  3. 这样就可以在Sublime Text中使用终端了
